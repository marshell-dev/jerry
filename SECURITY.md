# Security Policy

If you believe you have found a security issue in `jerry`, please do not open a public issue first.

Report it privately through GitHub Security Advisories for this repository or contact the maintainer directly if a private reporting channel is available.

Please include:

- a clear description of the issue
- affected versions or commit range
- reproduction steps or proof of concept
- suggested mitigation if available

Security reports will be acknowledged and triaged as quickly as possible.
